To install the TensorReg Toolbox for MATLAB:

1. Unpack the compressed file. In a linux environment, for example, this can
   be done from the command line via:

     unzip tensorreg_toolbox_0.0.1.zip

   *or*

     gunzip -c tensorreg_toolbox_0.0.1.tar.gz | tar xvf -

   This should create a directory named *tensorreg_toolbox_0.0.1*.

2. Rename the root directory from *tensorreg_toolbox_0.0.1* 
   to *tensorreg_toolbox*.

3. Start MATLAB.

4. Within MATLAB, cd to the *tensorreg_toolbox* directory and execute the
   following commands:

     addpath(pwd) %<-- Add the TensorReg toolbox to the MATLAB path
     savepath %<-- Save for future MATLAB sessions
